An app use to simulate different weathers, such as sunny, rainny...

Getting Started
  • To use the WaterMove API, first add the jQuery script.
    • Constructor
    new WaterMove()
    create an WaterMove object with all default arguements be 0s.
    API Methods
    .getRainTime()
    No argument needed, return the total time in second and the total precipitation for rainy as a string.
    .getSnowTime()
    No argument needed, return the total time in second and the total precipitation for snowy as a string.
    .getHailTime()
    No argument needed, return the total time in second and the total precipitation for hail as a string.
    .clearInfo()
    No argument needed, set the obeject to default state.
    .makeRain(amount, speed)
  • A rainy animation with the amount arguments shows the amount of rain in a moment and the speed of rain drop.
  • amount: the amount of rain in a moment, must be non-negative integers. 1 is the smallest amount.
  • speed: the speed of rain drop, must be positive integers. 1 means fastest speed.
    • .makeSnow(amount, speed)
    similar as .makeRain(amount, speed)
    .makeHail(amount, speed)
    similar as .makeRain(amount, speed)
    . makeSun(amount, direction, speed, radius, height_diff)
  • The sun animation with the amount, direction, speed, radius and height_diff.
  • amount: use to control whether the sun shows in web, only 0 shows false.
  • direction: left/right, show the movement direction of the sun.
  • speed: the speed of sun movement, must be positive integers.
  • radius: the radius of the sun in px, should be positive.
  • height_diff: the difference in vh between the highest and lowest position of sun, should be non-negative.
    • .makeFog(amount,direction, i1, s1, i2, s2,i3,s3, i4,s4, i5, s5)
  • The fog animation.
  • amount: use to control the number of images use to make fog, can add at most 5 images.
  • direction: left/right, show the movement direction of the fog.
  • is: the index of the images wanted, should be integers between 1-5.
  • ss: the speed of the corresponding images, should be Value should be positive integers. 1 means fastest speed.
    • .makeCloud(amount,direction, i1, s1, i2, s2,i3,s3, i4,s4, i5, s5)
    similar as .makeFog(amount,direction, i1, s1, i2, s2,i3,s3, i4,s4, i5, s5)
